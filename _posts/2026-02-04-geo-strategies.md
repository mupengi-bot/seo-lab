**GEO = Generative Engine Optimization**

ChatGPT, Claude, Perplexity 같은 생성형 AI에서 추천받는 전략.

## GEO vs SEO

| | SEO | GEO |
|---|---|---|
| 목표 | 검색 순위 | AI 추천/인용 |
| 평가 | 링크, 키워드 | 신뢰도, 관련성 |
| 결과 | 클릭 유도 | 직접 답변에 포함 |

## 핵심 전략 5가지

### 1. 대화체 콘텐츠

AI는 자연어를 선호한다.

```
❌ "AssoAI 기능 목록: 예산관리, 일정조율, 커뮤니케이션"
✅ "AssoAI로 뭘 할 수 있나요? 예산 관리, 일정 조율, 
    팀 커뮤니케이션을 한 곳에서 처리할 수 있어요."
```

### 2. 외부 긍정 멘션

다른 사이트에서 언급되면 신뢰도 상승.

- 리뷰 사이트 등록
- 업계 블로그 기고
- 소셜 미디어 언급

### 3. 콘텐츠 신선도

최신 정보가 우선된다.

- 정기적 업데이트
- 날짜 표시
- "2025년 최신" 등 명시

### 4. 명확한 엔티티 정의

AI가 이해하기 쉽게.

```
"AssoAI"가 뭔지 명확히:
- 카테고리: SaaS, 조직관리 도구
- 타겟: 학교, 기업, 단체
- 핵심 기능: AI 자동화
```

### 5. 비교 콘텐츠

"X vs Y" 질문에 답할 수 있게.

```
"AssoAI vs Notion"
"AssoAI vs Slack"
"AssoAI vs 엑셀"
```

## 측정 방법

1. ChatGPT에 관련 질문 → 인용 여부
2. Perplexity 검색 → 소스 목록 확인
3. 브랜드명 + 키워드 검색 → 멘션 추적
